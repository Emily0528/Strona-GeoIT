Podstawowe Informacje o Microsoft SQL Server

Wprowadzenie do MS SQL Server

Microsoft SQL Server (MS SQL) to relacyjny system zarządzania bazami danych (RDBMS) stworzony przez firmę Microsoft. Jest jednym z najpopularniejszych systemów bazodanowych używanych na świecie, znanym ze swojej wydajności, skalowalności i integracji z innymi produktami Microsoft, takimi jak Windows Server, Azure i Visual Studio.

Edycje MS SQL Server

MS SQL Server jest dostępny w kilku edycjach, które różnią się funkcjonalnością i ceną, aby sprostać różnym potrzebom użytkowników:

  • SQL Server Express: Bezpłatna edycja, przeznaczona do małych aplikacji i nauki.
  • SQL Server Developer: Darmowa edycja z pełną funkcjonalnością SQL Server, przeznaczona do tworzenia i testowania aplikacji.
  • SQL Server Standard: Edycja przeznaczona dla małych i średnich firm, oferująca podstawowe funkcje zarządzania bazami danych.
  • SQL Server Enterprise: Najbardziej zaawansowana edycja, oferująca pełen zestaw funkcji dla dużych organizacji i krytycznych aplikacji.

Architektura MS SQL Server

  • Instancja SQL Server: Każdy uruchomiony egzemplarz SQL Server to osobna instancja. Można uruchomić wiele instancji na jednym serwerze fizycznym lub wirtualnym.
  • Baza danych: Zbiór tabel, widoków, procedur składowanych, funkcji, indeksów i innych obiektów.
  • Tabela: Podstawowy obiekt do przechowywania danych w bazie danych. Tabele składają się z wierszy i kolumn.
  • Indeks: Struktura danych poprawiająca szybkość wyszukiwania i operacji na tabelach.
  • Widok:Wirtualna tabela, która jest wynikiem zapytania do jednej lub więcej tabel.
  • Procedury składowane:Zestaw instrukcji SQL, które mogą być zapisane i uruchamiane na serwerze.
  • Funkcje:Funkcje zdefiniowane przez użytkownika, które zwracają pojedyncze wartości lub tabele.

Przykładowe Zapytania

Tworzenie bazy danych:

CREATE DATABASE NewDB;

Tworzenie tabeli:

CREATE TABLE Employees (
EmployeeID INT PRIMARY KEY,
FirstName NVARCHAR(50),
LastName NVARCHAR(50),
BirthDate DATE
);

Wstawianie danych:

Wstawianie danych:
VALUES (1, 'John', 'Doe', '1980-01-01');

Zapytanie SELECT:

SELECT * FROM Employees;

Aktualizacja danych:

UPDATE Employees
SET LastName = 'Smith'
WHERE EmployeeID = 1;

Usuwanie danych:

DELETE FROM Employees
WHERE EmployeeID = 1;
